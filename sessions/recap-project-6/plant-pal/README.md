# PlantPal

## Description

PlantPal is an interactive platform designed to help users care for their plants effectively. It allows users to track their plants’ health, receive reminders for watering and fertilising, and access a comprehensive database of plant care information.

### Main Features

- **Plant List:** Browse a well-organised collection of plants.
- **Plant Details:** View detailed information about each plant.
- **Mark Plants as Owned:** Keep track of the plants you own.
- **Create Plant:** Add new plants, enriching the plants catalogue.
- **Delete Plant:** Remove outdated or unwanted plants.
- **Update Plant:** Update and modify existing plant details.
- **Filter Plants:** Filter plants based on various criteria.

### Additional Feature Inspiration

These features are optional and should serve as a conversation starter for upcoming features of the app.

- **Upload Images for Plants:** Personalise your collection by adding images for each plant.
- **Organise by Location:** Create collections based on the plants’ locations, such as garden or living room.
- **Set Custom Reminders:** Maintain a regular care schedule with personalised reminders.
- **Track Daily Streaks:** Maintain consistent plant care habits by tracking daily activities.
- **Identify Plants with AI:** Use AI to identify plants and receive care tips.
- **Get Local Weather Alerts:** Receive care instructions and alerts based on local weather conditions.
