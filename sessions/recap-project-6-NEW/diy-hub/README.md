# DIY Hub

## Description

The DIY Hub enables users to create, organise, and plan their DIY projects. It provides a platform for users to track their progress, organise materials, and store detailed information about each project.

### Main Features

- **[Database Connection](./user-stories/0_database-connection.md):** Set up MongoDB connection and data models.
- **[Projects List](./user-stories/1_projects-list.md):** Browse a well-organised collection of DIY projects.
- **[Project Details](./user-stories/2_project-details.md):** View detailed information about each project.
- **[Create Project](./user-stories/3_create-project.md):** Add new projects, enriching the projects catalogue.
- **[Update Project](./user-stories/4_update-project.md):** Update and modify existing projects.
- **[Delete Project](./user-stories/5_delete-project.md):** Remove outdated or unwanted projects.

### Additional Features Inspiration

These features are optional and should serve as a conversation starter for upcoming features of the app.

- **[Bookmark Project](./user-stories/additional-tasks/bookmark-project.md):** Save and showcase your favourite projects.
- **[Filter Projects](./user-stories/additional-tasks/filter-projects.md):** Filter projects by various criteria.
- **Upload Project Images:** Add images to projects to provide visual aids and inspiration.
- **Add Project Notes:** Include private notes to projects to keep track of ideas, modifications, and personal insights.
- **Track Progress:** Monitor the progress of projects with status updates (e.g., planning, in progress, completed).
- **Search Projects:** Use a search bar to quickly find projects by name or keyword.
- **Rate Projects:** Rate projects on a scale from 1 to 5 stars.
