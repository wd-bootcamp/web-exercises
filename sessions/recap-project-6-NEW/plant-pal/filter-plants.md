# Filter Plants

## Value Proposition

**As a user**

**I want to** filter plants based on their light needs,

**so that** I can quickly find plants that match the light conditions in my home or garden.

## Acceptance Criteria

- The plant list includes a section with filter options.
- Users can filter plants by light needs.
- Only one light need filter can be selected at a time.
- Selecting a new light need filter automatically deselects the previously selected filter.
- The applied filter is clearly displayed on the plant list, allowing users to see which filter is currently active.
- Users can clear all filters with a single click, returning the list to its unfiltered state.
- The plant list updates in real-time as filters are applied or removed.
- If no plants match the selected filter criteria, a message is displayed indicating that no plants were found.
- The filter interface is intuitive and easy to use, ensuring that new users can easily understand how to apply and clear filters.
