# FlipWise

## Description

FlipWise is an intuitive platform designed for effective learning and memorisation through the use of digital flashcards. It allows users to create, manage, and study flashcards efficiently, fostering better retention of information through interactive and engaging methods.

### Main Features

- **Flashcard List:** Browse a well-organised collection of flashcards.
- **Flip Flashcard:** Click on a flashcard to flip it and view the answer.
- **Mark as Correct:** Mark the flashcard as correctly answered.
- **Create Flashcard:** Add new flashcards, enriching the flashcards catalogue.
- **Delete Flashcard:** Remove outdated or unwanted flashcards.
- **Update Flashcard:** Update and modify existing flashcards.
- **Collections List:** Browse a well-organised collection of flashcard sets.

### Additional Features Inspiration

These features are optional and should serve as a conversation starter for upcoming features of the app.

- **Add Collection:** Organise flashcards by creating new collections.
- **Delete Collection:** Remove unnecessary collections to stay organised.
- **Adaptive Learning Algorithm:** Improve your study efficiency with an algorithm that shows correctly answered flashcards less frequently.
- **Daily Streaks:** Track daily usage to maintain consistent study habits.
- **Custom Reminders:** Set reminders to maintain a regular study schedule.
- **Quiz Mode:** Test your knowledge with a quiz mode that lets you answer a set of flashcards within a limited time.
- **AI-Generated Flashcards:** Automatically create flashcards from your learning materials using AI.
