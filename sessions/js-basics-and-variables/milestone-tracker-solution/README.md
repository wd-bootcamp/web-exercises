# JS Basics and Variables: Milestone Tracker

## Tasks

In this challenge, you’ll create a program to calculate interesting statistics based on how many years you’ve been practicing a skill (coding, guitar, drawing, a language—anything).

Please switch to the `index.js` file to begin the exercise, where you’ll find all instructions.

Hint: To test your work, run the code in the terminal by replacing `<years>` with the number of years:

```bash
node index.js <years>
// e.g.
node index.js 3
```

## Notes

- You only have to touch the `index.js` file.

> ❗️ This project is JavaScript with Node only. Run the code in the terminal as shown above.
